# Anchor

The Anchor element provides a click interaction on its child, that
when clicked sets the window location to the specified `href` URL.

<Story name="Default">
  <div className={styleClasses.textBox}>
    <div className={styleClasses.textboxitem}>
      Here's{' '}
      <Anchor href={window.parent.location}>
        a link to a page that's been visited
      </Anchor>
      , and here's <Anchor href={`https://www.example.com/${uuidv4()}`}>
        a link to a page that has not
      </Anchor>.
    </div>
  </div>
</Story>

```tsx
// prettier-ignore
const TwoAnchors = () => (
  <Paragraph>

      a link to a page that has not
    </Anchor>
    .
  </Paragraph>
)
```

Clicking the **color scheme button** in the toolbar ☝ will alter the color
of the link text.
