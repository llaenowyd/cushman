# Anchor

The Anchor element provides a click interaction on its child, that
when clicked sets the window location to the specified `href` URL.

```tsx
<Paragraph>
  <Anchor href="survey.html">Click here</Anchor> to see the latest results!
</Paragraph>
```

Clicking the **color scheme button** in the toolbar ‚òù will alter the color
of the link text.
