# Button

The Button element provides a means for the user to signal the
impulse to act, to take some action or that the action be taken
by the software, to begin an event.

<Story name="Default">
  <DefaultExample />
</Story>

```tsx
import { Button } from '@a110/rito'

// prettier-ignore
const PlusMinusControl = ({ setIntegerState }) => (
  <>
    <Button
      label="Increment"
      onClick={() => setIntegerState(i => 1 + i)}
    />
    <Button
      label="Decrement"
      onClick={() => setIntegerState(i => -1 + i)}
    />
  </>
)
```

Clicking the **color scheme button** in the toolbar ☝ will alter the color
of the button.
